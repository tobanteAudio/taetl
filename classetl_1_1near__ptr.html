Detailed Description

template<class Type, typename StorageType = uint16_t>
class etl::near_ptr< Type, StorageType >

Compressed pointer to specified size. Intended to be used as a drop in replacement for native pointers.

Only useful if the memory range starts at 0, since the near_ptr doesn't use a base address offset.
