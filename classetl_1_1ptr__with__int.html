Detailed Description

template<typename T, size_t PointerBits, size_t IntBits, typename IntType = size_t, typename StorageType = uintptr_t>
class etl::ptr_with_int< T, PointerBits, IntBits, IntType, StorageType >

Wraps a pointer and an integer value. Uses as much space as sizeof of the StorageType template parameter.

Template Parameters
TThe value_type of the wrapped pointer.
PointerBitsHow many bits to use for storing the pointer.
IntBitsHow many bits to use for storing the integer.
IntTypeThe type of integer to be stored.
StorageTypeThe underlying storage for both pinter & integer.
