Detailed Description

template<typename Key, etl::size_t Capacity, typename Compare = etl::less<Key>>
class etl::static_set< Key, Capacity, Compare >

etl::static_set is an associative container that contains a sorted set of unique objects of type Key. Sorting is done using the key comparison function Compare.
