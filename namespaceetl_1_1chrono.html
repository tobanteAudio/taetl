template<typename Rep , typename Period , typename = etl::enable_if_t<numeric_limits<Rep>::is_signed>>
constexpr auto etl::chrono::abs ( duration< Rep, Period >  d) -> duration<Rep, Period>

Returns the greatest duration t representable in ToDuration that is less or equal to d. The function does not participate in the overload resolution unless ToDuration is an instance of etl::chrono::duration.

Returns the absolute value of the duration d. Specifically, if d >= d.zero(), return d, otherwise return -d. The function does not participate in the overload resolution unless etl::numeric_limits<Rep>::is_signed is true.
